**这是一个大一学生基于easyx图形库写的c++贪吃蛇小项目**   

个人初步想法，实际实现根据[b站视频](https://www.bilibili.com/video/BV1HL4y1A7qX?t=432.2)实现  

1.使用面向对象程序设计的思路，把蛇设计为一个类，然后各种在类中内置各种方法：转弯，吃豆后变长等，以及蛇身数组等属性  

2.每一帧更新，检测是否撞墙或者是否吃到豆，然后调用蛇对象对应的方法  

3.如何实现移动：没有很好的想法，目前猜测是将蛇身体的部分保存在一个数组里面，然后每一帧更新一下尾部删掉一个，头部朝着记录的方向增加一个  

4.如何实现转弯？每一帧检测按键，方向更新为按键方向  

5.目前没什么想法来实现自身碰撞的检测，难道是遍历蛇身数组每个吗？会不会有更好的数据结构和方法。  

---  
### 目前实现的主要功能有：  
1.贪吃蛇基本逻辑  

2.自身碰撞死亡  

3.死亡后来到结束页面

4.结束页面显示本次长度和历史最佳长度  

5.结束后按'e'退出游戏
